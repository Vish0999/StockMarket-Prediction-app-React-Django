# 📈 Stock Market Prediction App

A full-stack web application for **stock market analytics, portfolio management, and crypto tracking** built using **Django REST API and React**.

---

# 🚀 Tech Stack

## Backend
- Python
- Django
- Django REST Framework

## Frontend
- React (Vite)
- Axios
- JavaScript

## Deployment
- Gunicorn
- Nginx
- Linux Server (Azure VM)

---
